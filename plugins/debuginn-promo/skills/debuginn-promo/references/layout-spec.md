# Layout Spec

Use this reference when you need concrete layout guidance for the square composition.

## Recommended Structure

- Canvas size suggestion: `1080 x 1080`
- Background: `#FFFFFF`
- Left content box: `500-560px` wide
- Right text box: `360-420px` wide
- Safe margin: `72-96px`

## Visual Placement

- Center the screenshot vertically.
- If the screenshot is tall, preserve aspect ratio and let it occupy most of the left column height.
- Optional: use a subtle shadow under the screenshot only if it improves separation.
- Optional screenshot corner radius: `24-36px`

## Typography

- Title:
  - Weight: bold or semibold
  - Color: `#111111`
  - Size suggestion: `54-76px`
- Subtitle:
  - Weight: regular
  - Color: `#666666`
  - Size suggestion: `26-34px`
  - Line height: `1.4-1.6`

## Chinese Copy Guidance

- Title should be short and memorable.
- Subtitle should explain function, benefit, or scenario.
- Avoid long paragraphs.

## Example

- Title: `记账更清楚`
- Subtitle: `一眼看懂每日支出和预算变化`
